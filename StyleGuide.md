# InquizIt Style Guide

## Colors

- Background: `#083048`
- Menu Background: `#07263B`
- Accent: `#FB9E05`
- Text Color: `#FEFEFE`

---

## Fonts

- Title
  - Font: `Secular One`
  - Size: `40pt`
- Subtitle
  - Font: `Secular One`
  - Size: `24pt`
- Paragraph
  - Font: `Lato Regular`
  - Size: `22pt`

---

## Spacing

Absolute spacing is based on 1280x720 window size

- Margins: `20px`
- Padding: `40px`
- Top Padding: `20px`

---

## Design Extras

- Title should be centered at the top of a menu
- Subtitles should have an accent line 4px below them
